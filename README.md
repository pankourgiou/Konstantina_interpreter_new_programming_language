Αυτός είναι ο Interpreter για μία νέα γλώσσα προγραμματισμού την Κωνσταντίνα που σχετίζεται με αρχεία .Κωνσταντίνα, για να τρέξει ο interpreter θα πάτε στο command prompt και θα τρέξετε το πρόγραμμα από το φάκελο στον οποίο είναι αντεγραμένο και σωσμένο
άρα αν είναι σωσμένο στο Desktop, θα πατήσετε cd Desktop για να κατευθυνθείτε στον φάκελο αυτό, και μετά θα τρέξετε : python Κωνσταντίνα_interpreter.py  εκεί μπορείτε να πληκτρολογήσετε βοήθεια, όπου δίνονται παραδείγματα εντολών.
